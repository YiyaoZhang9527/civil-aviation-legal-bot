## 根因定位

对 95 条 unverifiable claims 逐题追查，定位四个根因：

| 根因 | 影响 | 机制 |
|------|------|------|
| Reranker 阈值过高 | ~30 条 | `RERANKER_MIN_SCORE=0.1` 把正确结果全过滤掉，适航指令等短条文最高才 0.0996 |
| Level 1 单法规垄断 | ~30 条 | 运输类旋翼适航规定占 8/15 个章节名额，适航指令规定只有 1 个 |
| 证据文本截断 | ~25 条 | 评测脚本 `text[:800]` 截断，121.25 条原文 ~1500 字被切半 |
| 知识库缺民航局公告 | ~13 条 | 禁带物品清单不在 129 部法规中 |

关键发现：Q24 适航指令在 Level 0 排第 1，Level 1 也排第 1，但 Level 2 候选全被运输类旋翼适航规定占满，reranker 又把剩下的全过滤了。

## 改动清单

### 1. Reranker 回退机制（retrieval.py:608-621）

**问题**：reranker 对短文本打分偏低，全部低于 0.1 阈值 → 零结果返回
**修复**：reranker 全部过滤时，回退到 prerank raw top_k（未精排的原始候选）

```python
# 修复前
if config.RERANKER_MIN_SCORE > 0:
    reranked = [item for item in reranked if item.get("_rerank_score", 0) >= config.RERANKER_MIN_SCORE]
deduped = [item["_evidence"] for item in reranked if "_evidence" in item]

# 修复后
if config.RERANKER_MIN_SCORE > 0:
    filtered = [item for item in reranked if item.get("_rerank_score", 0) >= config.RERANKER_MIN_SCORE]
else:
    filtered = reranked
if filtered:
    deduped = [item["_evidence"] for item in filtered if "_evidence" in item]
else:
    deduped = prerank[:top_k]  # 回退：保底用 raw 排序
```

### 2. Level 1 章节法规多样性（tree_retrieval.py:246-258）

**问题**：运输类旋翼适航规定占 8/15 个章节，适航指令规定只有 1 个，Level 2 候选全来自同一法规
**修复**：Level 1 章节选择后加法规多样性控制，每部法规最多 4 个章节

```python
# 新增：法规多样性控制
per_law = getattr(config, 'TREE_CHAPTER_PER_LAW', 4)
if per_law > 0:
    law_count: dict[int, int] = {}
    diverse = []
    for idx, score in chapter_set:
        law_idx = int(self._chapter_to_law[idx])
        cnt = law_count.get(law_idx, 0)
        if cnt < per_law:
            diverse.append((idx, score))
            law_count[law_idx] = cnt + 1
    chapter_set = diverse
```
