# 评测方法论问题：45.5% 不是幻觉率

> 日期：2026-06-02
> 触发：审查 tests 文件夹文档和代码后发现评测指标存在根本性缺陷

---

## 1. 问题发现

当前 30 题测试的"幻觉率 45.5%"（157/345 unsupported）**测的不是答案幻觉，而是检索覆盖率缺口**。

## 2. 当前评测流程

```
test_30questions.py → run_question() → orch.answer(question) → AnswerResult
  → result.citations → 统计 supported/partial/unsupported
```

`result.citations` 来自 `CitationAgent.verify()`：

```
question + legal_issues
  → LLM 提取 claims（从问题中提取法律主张）
  → 每条 claim × 每条 evidence → Cross-Encoder 打分
  → score ≥ 0.25 → supported
  → score ≥ 0.15 → partial
  → score < 0.15 → unsupported
```

**关键**：这是在判断"检索到的证据是否支持问题的主张"，不是在判断"答案是否编造了内容"。

## 3. "unsupported"的实际含义

| 场景 | 当前评测 | 应有的判定 |
|------|----------|-----------|
| 检索没找到相关证据 | unsupported | 检索失败（非幻觉） |
| 系统回答"无法确定" | unsupported | 正确拒答（非幻觉） |
| 答案正确但证据 CE 分低 | unsupported | 正确答案（非幻觉） |
| 答案编造了不存在的法条 | 可能 supported | 真正的幻觉 |
| 答案歪曲了法条含义 | 可能 supported | 真正的幻觉 |

## 4. 实际案例

### Q03（92% "幻觉"）— 实际：正确拒答
- 问题：航空公司在什么情况下可以低于最低天气标准运行
- 答案：「根据现有证据，无法确定……因为提供的证据未包含相关规定」
- 判定：11/12 unsupported → 92% "幻觉"
- **实际**：系统正确地拒绝了回答，零幻觉

### Q28（75% "幻觉"）— 实际：正确答案
- 问题：中华人民共和国对领空享有什么权利
- 答案：「中华人民共和国对领空享有完全的、排他的主权」
- 引用：《民用航空法》第二条 — 完全正确
- 判定：9/12 unsupported → 75% "幻觉"
- **实际**：答案完全正确，但检索到的其他证据（民用航空法的无关条文）CE 分低

### Q20（83% "幻觉"）— 实际：正确拒答
- 问题：旅客行李丢失后航空公司如何赔偿
- 答案：「相关条款未提供」
- **实际**：系统承认无法回答，零幻觉

### Q19（83% "幻觉"）— 实际：给了实质性回答，需人工判断
- 问题：航空公司拒载旅客的合法理由有哪些
- 答案给出了具体理由，但检索到的证据不理想
- **实际**：可能包含部分正确信息和部分推断，需独立评测

## 5. 我们的指标 vs Stanford 指标 vs 商用评测

| 维度 | 我们的评测 | Stanford 评测 | 商用系统评测 |
|------|-----------|--------------|-------------|
| 测什么 | 证据 CE 得分 ≥ 0.25 | 答案是否包含编造内容 | 答案引用是否真实+正确 |
| unsupported 含义 | 检索没找到高相关证据 | 答案引用了不存在的案例 | 答案歪曲或编造 |
| 拒答处理 | 拒答仍判 unsupported（惩罚） | 拒答不算幻觉（中性） | 拒答不算幻觉 |
| 正确答案+低分证据 | 判为"幻觉" | 不算幻觉 | 不算幻觉 |
| 判定方式 | 自动（Cross-Encoder） | 人工专家评审 | 人工+自动化混合 |
| 自评 vs 独立 | **系统自评** | **独立评审** | **独立评审** |

## 6. 核心结论

1. **45.5% 是检索覆盖率缺口，不是幻觉率**
2. **真正的答案级幻觉率我们根本没测**
3. 系统自评有循环论证问题：用同一个系统的 CitationAgent 检查自己的输出
4. 要对标商用系统，需要一个**独立于系统自身**的答案级评测方法

## 7. 需要解决的问题

如何构造一个独立的评测方法，能够：
- 区分"检索失败"和"答案幻觉"
- 正确处理拒答（不算幻觉）
- 检测答案中编造的法条引用
- 检测答案中对法条含义的歪曲
- 可自动化运行，不需要每题都人工评审
